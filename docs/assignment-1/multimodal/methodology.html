Protocol comparison

Method decision logic

  • Zero-shot tests whether pretrained vision-language alignment is already sufficient for the chosen 10-class Food101 task.
  • Few-shot tests whether a lightweight supervised head can improve task fit without expensive backbone finetuning.
  • Both branches share the same CLIP image encoder, preprocessing stack, and test split so the comparison isolates head strategy.
  • The saved runs show zero-shot still ahead, which means future work should focus on stronger adaptation than a plain linear probe.
