Overview

Dataset scope and experiment framing

The full Food101 dataset contains 101 classes with 75,750 training images and 25,250 test images. For this assignment, the multimodal track narrows the problem to 10 food categories and then creates a balanced few-shot train/dev split before evaluating on the full held-out test subset.

Split recipe

Component What is included
Original dataset Food101 with 101 classes, distributed as 75,750 train and 25,250 test images.
Selected classes 10 food categories aligned with the CLIP zero-shot vs few-shot experiment.
Filtered subset 10,000 images total, with exactly 1,000 images available for each chosen class.
Few-shot train split 1,280 images total, sampled as 128 images per class using seed 42.
Few-shot dev split 200 images total, sampled as 20 images per class from the same filtered train pool.
Test split 2,500 images total, using the full held-out test subset of the 10 selected classes.

Primary findings

  • The selected 10-class problem removes label-frequency bias because every chosen class starts with 1,000 images.
  • The experiment subset is dominated by held-out test data, which is appropriate because final reporting happens on that split.
  • Image aspect ratios vary from 0.56 to 2.14, so resize-and-center-crop choices materially affect retained visual context.

Actionable next step

  • Keep class-balanced sampling for all few-shot runs so performance changes reflect representation learning rather than skew.
  • Use preprocessing that preserves framing better than a naive square crop, especially for elongated dishes and plated food scenes.
  • Retain the full test subset because it provides enough examples per class to expose failure patterns in confusion analysis.

Composition

Split share of the experiment subset

The active subset contains 3,980 images, and almost two thirds belong to the held-out test split.

Test accounts for 62.8% of the active subset, few-shot train for 32.2%, and few-shot dev for 5.0%.

Geometry

Image size and aspect ratio variation

The dataset is not geometrically uniform. Width and height are often close to 512, but aspect ratios still vary enough to matter.

Width ranges from 287 to 512 pixels, height ranges from 239 to 512 pixels, and median area is 262,144 pixels.

Width-height scatter

The diagonal reference line shows that many images are nearly square, but there is still meaningful spread above and below that line.

Mean width is 494.3 pixels, mean height is 473.0 pixels, and the average aspect ratio is 1.075 with standard deviation 0.233.

Samples

Class gallery with intra-class variety

Three examples per class make the intra-class spread more obvious, especially for food types that appear in different lighting conditions and presentation styles.

This diversity justifies comparing learned few-shot heads against prompt-averaged zero-shot CLIP embeddings.

Interpretation

What the EDA implies for modeling

Balanced labels The evaluation should focus on semantic confusion and representation quality, not imbalance correction.
Diverse framing Preprocessing should preserve composition where possible because food context often extends beyond the central crop.
High visual overlap Classes such as pizza, grilled cheese sandwich, and hamburger still require strong texture and shape cues despite clean label balance.

Evidence

Numerical summary

  • The selected subset contains 10,000 images total, with 1,000 images per chosen class.
  • The active multimodal experiment uses 3,980 images after constructing the balanced few-shot train and dev subsets.
  • Test contributes 2,500 images, which is 62.8% of the active experiment subset.
  • Aspect ratio ranges from 0.561 to 2.142, confirming that the dataset is not uniformly square despite many 512-pixel images.
  • The mean image area is 233,143 pixels, while the median is 262,144 pixels, indicating many images cluster around 512 × 512.
