Advanced Experiment Template

1. Robustness Question

Target shifts, deployment assumptions, and robustness metric.

2. Shift Profile

Noise, style/domain, class prior, and label sparsity setup.

3. Hypotheses

Pre-registered expectations for each defense strategy.

4. Experiment Matrix

Scenario, model, defense method, metric, and notes per run.

5. Results Dashboard

IID score, OOD score, robustness gap, and calibration error.

6. Sensitivity Analysis

Learning rate, regularization, and augmentation sensitivity.

7. Risk and Error Analysis

Spurious correlations, overconfidence, and failure examples.

8. Conclusion

Most robust setup, tradeoffs, and next iteration.
