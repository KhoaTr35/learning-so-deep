Advanced Experiment Template

1. Problem Statement

Task objective, primary metric, success threshold, and compute/data constraints.

2. Dataset Card

Dataset source, split, class count, known bias, and license.

3. Hypotheses

Define H1 to H3 before running experiments.

4. Experiment Matrix

Track model variants, hyperparameters, and rationale per run.

5. Results Dashboard

Report accuracy, precision, recall, F1, inference latency, and model size.

6. Error Analysis and Ablation

Document confusion patterns, hard samples, and component impact.

7. Reproducibility Checklist

Seed, environment versions, commands, artifacts, and checkpoints.

8. Conclusion

Best model, failure modes, and next iteration plan.
